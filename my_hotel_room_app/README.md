# my_hotel_room_app (Room Booking Application)

A Flutter application for booking hotel rooms.

## Description

This application allows users to browse available rooms and make reservations by selecting check-in and check-out dates.
It connects to a backend API built with Node.js and Prisma to manage booking data and prevent duplicate reservations.

## Main Features

- View available rooms

- Select check-in and check-out dates

- Book one or multiple rooms

- Prevent duplicate bookings (date validation)

- API integration using Dio with interceptors

- Real-time UI updates using GetX state management

## Technologies Used

-Flutter (Dart)

-GetX (State Management)

-Dio (HTTP Client)

## Getting Started

1. Install dependencies

```bash
flutter pub get
```

2. Run the application

```bash
flutter run 
```
